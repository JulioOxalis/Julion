What you get

Built for developers

Everything you need to snapshot, store, and restore any project — nothing you don't.

📦

julion seal

Compress your entire project — every file, directory, and checksum — into a single portable .on container in seconds. Works with any language or framework.

julion seal
☁️

julion deposit

Push your snapshot to Google Drive automatically. Organised under your JULION folder by repository and version. Your data stays in your own Drive, not our servers.

julion deposit
🔄

julion unseal

Restore any snapshot to any directory on any machine. Full file tree, exact checksums, zero dependencies beyond the CLI.

julion unseal
🔒

Checksum integrity

Every file inside a .on container carries a SHA checksum. On unseal, every file is verified — bit for bit identical to what was sealed. Tampering is caught immediately.

Built in
🌐

Web dashboard

Browse all your repositories and snapshots from the browser. Open any file directly in an inline Monaco editor without downloading the full archive.

dashboard
⚡

CLI-first, CI-ready

Fully scriptable. No GUI required, no cloud agents, no lock-in. Drop it into any CI pipeline with a single npm install and a couple of commands.

npm i -g julion
🔑

Google OAuth

Sign in with your Google account — no passwords to manage. Julion uses your own Drive storage so your snapshots are always under your control.

julion connect google
📂

Repository organisation

Snapshots are organised by repository name under JULION/<repo>/ in your Drive. Browse, fetch, or restore any specific version at any time.

julion fetch
🚀

Ultra pipeline

Run julion seal --ultra for maximum compression and full metadata collection — ideal for large projects or long-term archiving.

--ultra
