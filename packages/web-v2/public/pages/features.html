Features

Everything you need, nothing you don't.

📦

julion seal

Compress your entire project — files, metadata, checksums — into a single portable .on container in seconds. Works with any language or framework.

→ Snapshot in one command
☁️

julion deposit

Push your snapshot to Google Drive automatically. Organised under your JULION folder by repository and version. Your data stays in your own Drive.

→ Backed up to Drive
🔄

julion unseal

Restore any snapshot to any directory on any machine. Full file tree, exact checksums, zero dependencies beyond the CLI.

→ Restore anywhere
🔒

Checksum verification

Every file inside a .on container has a SHA checksum. On unseal, every file is verified — bit-for-bit identical to what was sealed.

→ Guaranteed integrity
🌐

Web dashboard

Browse all your repositories and snapshots from the browser. Open any file directly in an editor without downloading the full archive.

→ Works in any browser
⚡

CLI-first

Built for developers. The entire Julion workflow runs from your terminal — no GUI required, no cloud agents, no lock-in. Scriptable and CI-friendly.

→ Integrates with your workflow
