Compress every file, directory, and checksum into a single self-contained .on container in seconds.

Deposit to Drive

Push snapshots straight to your Google Drive. Organised by repository — your data stays in your own account.

julion deposit

Restore anywhere

Fetch and unseal any snapshot on any machine. Every file is checksum-verified on restore — bit for bit.

julion unseal

Checksum integrity

Every file inside a .on archive carries a SHA checksum. Tampered or corrupted files are caught on restore.
