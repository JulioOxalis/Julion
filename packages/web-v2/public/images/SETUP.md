# Julion — Required Image Assets

Place the following files in this directory (`public/images/`).
The logo appears in the navbar, footer, OG tags, and 404 page.

## Logo / OG images

| File              | Size     | Purpose                                 |
|-------------------|----------|-----------------------------------------|
| `julion-logo.png` | 512×512  | Navbar, footer, 404, structured data    |
| `julion-og.png`   | 1200×630 | Open Graph (Facebook, LinkedIn, Slack…) |

### OG image tips
- Background: `#090b11` (matches dark theme) or `#6b7bff`
- Centered logo + tagline "Seal, Store & Restore Your Projects"
- Tools: Figma, Canva, https://og-playground.vercel.app
